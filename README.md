# Projeto CoronaHotel
## Descrição
- Criação de uma API de hotel.
## Atributos:
  - Id
  - CNPJ
  - Nome
  - Capacidade
  - CEP
  - Rua
  - Bairro
  - Cidade
## Ações:
  - Listar todos hoteis
  - Buscar por um id específico
  - Remover determinado hotel pelo id
  - Criar novo hotel
  - Atualizar dados de um hotel
## Tecnologias
- Java
- JPA
- Spring bot
- Lombok
- MySQL
